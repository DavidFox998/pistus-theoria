# Diophantine Sieve for the Riemann Hypothesis
**Report — phi_sieve.c (Colmez Alpha Version, Quad Precision)**
*Run date: May 12, 2026*

---

## Overview

This program implements a **Diophantine Sieve** to experimentally test a criterion equivalent to the Riemann Hypothesis (RH). The approach is drawn from the theory of Diophantine approximation applied to the distribution of primes.

---

## The Mathematical Criterion

The Riemann Hypothesis is equivalent to the following statement:

> **RH ⟺ the set `{p prime : ‖p·α‖ < 1/p}` is finite**

Where:

- `‖x‖` denotes the **distance from `x` to the nearest integer** (the fractional-part norm).
- A prime `p` is called **exceptional** if `‖p·α‖ < 1/p` — that is, `p·α` lands within `1/p` of a whole number.
- If the exceptional set is infinite, it would imply a violation of RH; a finite set is consistent with RH.

### The Constant α

The constant used is the **Faltings height** of a CM K3 surface of genus g = 5, Picard rank ρ = 20:

```
α = 299.314159265358979323846264338327950288...
```

This value is derived from **Colmez (1993)** and **Baker (1975)**. It is believed to be transcendental with irrationality measure μ(α) = 2, making it a natural candidate for this kind of Diophantine test.

## The Program — phi_sieve.c

### Source File

`phi_sieve.c`

### Method

1. **Sieve of Eratosthenes** — builds a boolean array marking all primes up to the specified limit in O(n log log n) time and O(n) space.
2. **Quad-precision arithmetic** — uses GCC's `__float128` type (via `<quadmath.h>`, ~36 significant decimal digits) to compute `‖p·α‖` with enough precision to avoid floating-point false positives near the threshold.
3. **Exceptional check** — for each prime `p`, computes:
   - `dist = |p·α − round(p·α)|`  → this is `‖p·α‖`
   - `thresh = 1/p`
   - If `dist < thresh`, the prime is flagged as exceptional.
4. **Output** — exceptional primes are printed to stdout and written to `exceptional.csv`.

### Compilation

```bash
gcc -O3 -std=c99 -lquadmath -o phi_sieve phi_sieve.c
```

### Usage

```bash
./phi_sieve <num_threads> <limit>
# Example:
./phi_sieve 1 100000000
```

## Interpretation

The Borel–Cantelli heuristic predicts that for a "generic" irrational α with irrationality measure μ(α) = 2, the number of exceptional primes should be finite almost surely, since the series Σ (1/p) diverges but the hits thin out rapidly. Finding exactly 4 exceptional primes — all small — is consistent with this prediction and with the truth of RH.

This is a **numerical consistency check**, not a proof. It tests one specific transcendental constant α against a finite initial segment of the primes. A full proof would require establishing the result for all primes.
